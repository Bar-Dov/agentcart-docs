## Planned capabilities

- **Catalog:** Magento REST API product search and lookup
- **Checkout:** Guest cart API + order placement
- **Connection:** OAuth 2.0 integration token or API key

## Expected prerequisites

| Credential | Notes |
|------------|-------|
| Magento integration token | Created under System → Integrations |
| Store base URL | Including store code if multi-store |

## Timeline

Magento follows WooCommerce on the platform roadmap. [Contact us](https://app.agentcart.io) to express interest.
