# WooCommerce

import { Callout } from 'nextra/components'

<Callout type="info">
  WooCommerce support is on our roadmap. This page describes the planned integration.
</Callout>

## Planned capabilities

- **Catalog:** WooCommerce REST API product search and lookup
- **Checkout:** WooCommerce cart API + order creation
- **Connection:** API key + secret from WooCommerce → Settings → Advanced → REST API

## Expected prerequisites

| Credential | Scope needed |
|------------|-------------|
| WooCommerce API key | `read` for catalog, `write` for checkout |
| Store URL | Your WordPress site URL |

## Timeline

WooCommerce is the next planned platform after Shopify reaches full feature parity. [Contact us](https://app.agentcart.io) to be notified or to accelerate prioritization.
